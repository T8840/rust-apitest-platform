- Create API Requests with Axios
- Create Reusable Components

    - UPDATE Operation
    - DELETE Operation
    - READ Operation
- Testing the CRUD App
    - Create New Case
    - Update a Case
    - Get All Cases
    - Delete a Case
    - Test a Case
